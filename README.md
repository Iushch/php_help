###### Операции:
Математические операции: (+, -, /, *, %, **(степень))   
Строковые операции: (.(конкатенация), ``(выполнение команды))   
Операции сравнения: <, >, ==, !=, <=>(если 1 больше 1, второй -1, равны 0), ===;   
Логические операторы: || - or, && - and, xor(исключающее или)
